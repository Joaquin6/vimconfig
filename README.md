## My Vim config

Clone with:

    git clone git@github.com:moshen/vimconfig.git .vim

or

    git clone https://github.com/moshen/vimconfig.git .vim

Run `setup.sh`!

Here's a handy script for copy-paste:

    cd $HOME &&
    git clone https://github.com/moshen/vimconfig.git .vim &&
    cd .vim &&
    ./setup.sh
